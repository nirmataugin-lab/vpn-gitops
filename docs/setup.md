# Setup guide

## 1. Install packages on both VPS

```bash
apt update && apt install -y git jq gettext-base sing-box
```

## 2. Clone repo on both VPS

```bash
rm -rf /opt/vpn-gitops
mkdir -p /opt/vpn-gitops
cd /opt/vpn-gitops
git clone https://github.com/YOUR_USERNAME/vpn-gitops.git .
```

## 3. Install scripts

```bash
install -m 755 /opt/vpn-gitops/scripts/vpn-gitops-update.sh /usr/local/bin/vpn-gitops-update.sh
install -m 755 /opt/vpn-gitops/scripts/init-node.sh /usr/local/bin/vpn-gitops-init.sh
```

## 4. Set node type

VPS1:

```bash
echo vps1 > /etc/vpn-node-type
```

VPS2:

```bash
echo vps2 > /etc/vpn-node-type
```

## 5. Initialize node

Run the init script on each VPS. It generates keys automatically and prompts for peer information.

**Step 1 — Initialize VPS2 first** (generates cascade UUID and its keypair):

```bash
/usr/local/bin/vpn-gitops-init.sh
```

The script prints a block of values to share with VPS1. Copy them.

**Step 2 — Initialize VPS1** (generates client UUID and its keypair, prompts for VPS2 values):

```bash
/usr/local/bin/vpn-gitops-init.sh
```

Paste the VPS2 values when prompted.

## 6. First deploy

```bash
/usr/local/bin/vpn-gitops-update.sh
```

## 7. Auto-update every minute (cron)

```bash
(crontab -l 2>/dev/null; echo '* * * * * /usr/local/bin/vpn-gitops-update.sh >/var/log/vpn-gitops.log 2>&1') | crontab -
```

## Required variables

### VPS1

| Variable | Source |
|----------|--------|
| `CLIENT_UUID` | Auto-generated by init |
| `CASCADE_UUID` | Generated on VPS2, pasted during init |
| `REALITY_SERVER_NAME` | Entered during init |
| `VPS1_REALITY_PRIVATE_KEY` | Auto-generated by init |
| `VPS1_REALITY_SHORT_ID` | Auto-generated by init |
| `VPS2_IP` | Entered during init |
| `VPS2_PORT` | Entered during init |
| `VPS2_REALITY_SERVER_NAME` | Entered during init |
| `VPS2_REALITY_PUBLIC_KEY` | Generated on VPS2, pasted during init |
| `VPS2_REALITY_SHORT_ID` | Generated on VPS2, pasted during init |

### VPS2

| Variable | Source |
|----------|--------|
| `CASCADE_UUID` | Auto-generated by init |
| `VPS2_PORT` | Entered during init |
| `VPS2_REALITY_SERVER_NAME` | Entered during init |
| `VPS2_REALITY_PRIVATE_KEY` | Auto-generated by init |
| `VPS2_REALITY_SHORT_ID` | Auto-generated by init |
